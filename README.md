# AxeCode Frontend Application

AxeCode is a modern, high-performance web platform designed for developers and learners. Built with **React 19** and **Vite**, the application follows a strict **Clean Architecture**.

---

## 🚀 Quick Start

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Configuration
1. Create a `.env` file based on the environment variables mentioned in the code.
2. Ensure `VITE_API_BASE_URL` points to your backend instance.
